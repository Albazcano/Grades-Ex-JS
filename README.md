# Ejercicios-basicos-JS
Ejercicios básicos de Javascript

## Comenzando 🚀

Pide una nota (número). Muestra la calificación según la nota:
0-3: Muy deficiente
3-5: Insuficiente
5-6: Suficiente
6-7: Bien
7-9: Notable
9-10: Sobresaliente

### Pre-requisitos 📋

node.js & npm

### Instalación 🔧

>> npm install

## Stack 📌

HTML5
Javascript
